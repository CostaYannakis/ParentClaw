# Learning Assets

Learning assets are controlled child-facing activities, diagrams, manipulatives, worksheets, and interactive tools.

The child does not need unrestricted AI access to use an asset.

## Asset Folders

| Folder | Purpose |
| --- | --- |
| `number/` | Counting, place value, operations, estimation, number sense. |
| `fractions/` | Fractions, decimals, ratio, proportional thinking. |
| `geometry/` | Shape, space, measurement, angles, transformations. |

## Protocol

Read `LEARNING-ASSET-PROTOCOL.md` before creating assets.

Every asset should declare:

- what it teaches
- prerequisites
- age or level guidance
- materials or launch instructions
- what evidence it can produce
- how the parent should record that evidence
