# Cartesian Plane Midpoint And Distance Teaching Aid

## Purpose

This browser activity helps a learner place two points on a four-quadrant Cartesian plane and systematically solve:

- the midpoint of line segment AB
- the length of AB using Pythagoras
- the distinction between distance squared and actual distance

## Prerequisites

The learner should already know, or be ready to practise with parent support:

- x- and y-coordinates on a Cartesian plane
- positive and negative integers from -10 to 10
- basic substitution into formulas
- square numbers and square roots as the inverse of squaring

## Parent Setup

Open `cartesian-midpoint-distance.html` in a web browser. No internet connection or child AI access is required.

Use the default example first, then either drag points A and B on the graph or enter coordinates manually. The graph snaps to integer coordinates from -10 to 10 on both axes.

## Child Task

Ask the learner to:

1. Place point A and point B in any quadrants.
2. Read the coordinates aloud.
3. Draw or inspect the line segment AB.
4. Find the midpoint by averaging x-values and y-values separately.
5. Use the guide triangle to find horizontal and vertical changes.
6. Apply Pythagoras to find AB squared.
7. Square-root to find the actual length of AB.
8. Explain the method aloud without simply reading the answer panel.

## Scoring Or Interpretation

Treat the skill as stronger when the learner can do all of the following with little or no prompting:

- plot or read A and B correctly in any quadrant
- calculate the midpoint by averaging x and y separately
- identify horizontal and vertical changes accurately
- write the Pythagoras step as `AB² = Δx² + Δy²`
- state that `AB = √(AB²)` is the actual length
- check arithmetic before finalising the answer

If the learner only follows the answer panel, keep the skill as developing. If the learner explains the method on a new point pair, that is better evidence of transfer.
