# Cartesian Plane Gradient And Slope Teaching Aid

## Purpose

This browser activity helps a learner understand gradient/slope on a coordinate plane by connecting:

- horizontal change as run
- vertical change as rise
- gradient as `rise / run`
- positive, negative, zero, and undefined gradients

## Prerequisites

The learner should already know, or be ready to practise with parent support:

- x- and y-coordinates on a Cartesian plane
- positive and negative integers from -10 to 10
- basic fraction simplification
- the idea that a line can rise, fall, be horizontal, or be vertical

## Parent Setup

Open `cartesian-gradient-slope.html` in a web browser. No internet connection or child AI access is required.

Use the default example first: A=(1,2), B=(7,5). The run is 6, the rise is 3, and the gradient is `3/6 = 1/2`.

## Child Task

Ask the learner to:

1. Place or read points A and B.
2. Count the horizontal run.
3. Count the vertical rise.
4. Write `gradient = rise / run`.
5. Simplify the fraction if possible.
6. Say whether the gradient is positive, negative, zero, or undefined.
7. Explain the answer aloud from the diagram before reading the working panel.

## Base Training Sequence

1. **Visual count:** Use rising lines with positive coordinates, such as A=(1,2), B=(7,5).
2. **Fraction simplification:** Practise gradients such as `3/6`, `4/8`, `6/3`, and `-4/2`.
3. **Direction:** Compare rising and falling lines to build positive and negative gradient language.
4. **Special cases:** Show horizontal lines as gradient `0` and vertical lines as undefined.
5. **Formula transfer:** Move from counted rise/run to `(y2 - y1) / (x2 - x1)`.
6. **Retention mix:** Include one midpoint problem at the start of some sessions so midpoint stays active.

## Scoring Or Interpretation

Treat the skill as developing when the learner can count rise and run with support. Treat it as stronger when the learner independently:

- reads A and B correctly
- calculates `rise = y2 - y1`
- calculates `run = x2 - x1`
- writes `gradient = rise / run`
- keeps the sign correct
- names horizontal gradient as `0`
- names vertical gradient as undefined
