# Fractions Footpath Number Line

## Purpose

Help a student see a fraction number line as a simple straight footpath across equal blocks, from start `0` to school at `1 whole`, with the fraction labels placed at the stops between blocks.

## Prerequisites

- The student can count small whole numbers.
- The student has met simple fraction language such as halves, quarters, or thirds.

## Parent Setup

Open `fraction-footpath-number-line.html` in a browser. Sit with the student and select the number of equal blocks to show.

## Child Task

Ask the student to move the person along the footpath and name the fraction at each stop. The default setup uses five blocks, matching Audrey's lesson. Have the student explain how many blocks have been walked and how many are still ahead.

## Evidence Produced

- Fractions named at joins, such as `1/5`, `2/5`, `3/5`, and `4/5`.
- Whether the student treats the whole path as one whole.
- Whether the student can explain that the denominator counts equal blocks and the numerator counts blocks already walked.

## Scoring Or Interpretation

Look for reasoning, not speed. A strong response connects the visual blocks to fraction language: `3/4 means three out of four equal blocks from start to finish`.

## Memory Update Guidance

Record which denominators were used, whether the student counted blocks or stops, and any misconception such as treating `1/4` as the first block instead of the first stop after one block.
