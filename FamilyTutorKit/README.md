# Family Tutor Kit

Family Tutor Kit turns a parent's existing AI subscription into a structured tutoring coach.

The parent remains the tutor. The AI reads the learner memory, plans lessons, coaches the parent, records evidence, and recommends the next session.

## Core Idea

You already have a powerful AI. This kit gives it the structure, memory, and teaching tools needed to help you tutor your child properly.

## Folder Map

| Folder | Purpose |
| --- | --- |
| `skills/` | Portable AI instructions for tutoring tasks. |
| `memory-system/` | The lifetime learner memory specification, rules, and templates. |
| `learners/` | One private learner record folder per child. |
| `curriculum/` | Optional mappings from observed skills to school standards. |
| `assets/` | Activity, diagram, worksheet, and manipulative metadata. |
