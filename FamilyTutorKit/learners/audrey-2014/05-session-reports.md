### Session 2026-06-17-001

| Field | Value |

| Objective | Basic fractions on number lines using a five-block footpath |
| Evidence IDs | 2026-06-17-001 |
| Next step | Transfer the same five-block model to a plain 0 to 1 number line, then try a new denominator |

#### Summary

Audrey completed today's lesson on basic fractions on number lines. The successful model was a drawn footpath with five equal blocks, a person at the 0 start, and a school at the 1 destination.

#### What Worked

The footpath made the whole visible: one complete journey from 0 to 1. Splitting it into five blocks gave Audrey a concrete way to see fifths as equal steps along the same whole.

#### What Needs Review

Check whether Audrey can explain the fraction language: the bottom number names how many equal blocks make the whole, and the top number names how many blocks have been travelled.

#### Parent Follow-Up

Next session, draw the five-block path again and label 0/5, 1/5, 2/5, 3/5, 4/5, and 1. Then draw only a straight line from 0 to 1 and ask Audrey to place the same labels without the school picture.

Audrey completed a visual fractions task using a journey divided into sixths. The task connected being at 5/6 of the way through a walk with the remaining fraction and time left.
