# Lifetime Maintenance

A lifetime learner memory must stay useful for years without becoming a transcript dump.

The goal is durable educational memory: what the learner has understood, what remains fragile, what evidence supports that view, and what should happen next.

## Maintenance Rhythm

| Frequency | Action |
| --- | --- |
| After each session | Add evidence, update review queue, add a short session report. |
| Weekly | Check due reviews and remove stale priorities. |
| Monthly | Summarize repeated patterns in the learning map. |
| Each term | Review goals and archive excessive detail. |
| Yearly | Create a yearly summary and snapshot. |

## Yearly Summary

At the end of a school year or calendar year, create an archive file named like:

```text
archive/2026-summary.md
```

Include:

- major strengths
- secure skills
- developing skills
- recurring misconceptions
- effective teaching approaches
- accommodations that helped
- subjects or domains to prioritize next
- links to important evidence IDs

## What Stays Active

Keep these files concise and current:

- `00-profile.md`
- `01-learning-map.md`
- `03-review-queue.md`
- `04-goals.md`

Long historical detail belongs in:

- `02-evidence-log.md`
- `05-session-reports.md`
- `archive/`

## Promotion Rule

Promote a pattern from the evidence log to the learning map only when it is instructionally useful.

Useful patterns include:

- repeated independent success
- repeated prompted success
- repeated misconception
- reliable strategy preference
- known accommodation that improves learning

## Aging Rule

Older evidence is still valuable, but it should not overrule recent observed performance.

Use:

- old evidence to understand history
- recent evidence to choose the next lesson
- review checks to decide whether old secure skills remain secure

## Sensitive Notes

Sensitive notes should be:

- minimized
- kept instructionally relevant
- summarized when possible
- excluded from redacted AI-sharing copies unless needed

## Transition Points

Review the whole memory when the learner:

- changes school
- changes curriculum
- receives a new support plan
- starts a new school year
- has a major change in health, attention, motivation, or confidence
- begins working with a tutor or specialist

## Graduation Or Handover

When the learner is old enough, the memory can become a shared record.

Before sharing:

- remove parent-only notes
- separate sensitive records
- summarize long evidence logs
- explain the skill-state vocabulary
- keep the learner's dignity and consent central
