# Portable AI Skills

These files are reusable instructions for ChatGPT, Claude, Codex-style agents, and local AI assistants.

Attach or paste the relevant skill file with the learner memory files.

## Skill Map

| Skill | Use When |
| --- | --- |
| `assess-current-level.md` | You need to find a sensible starting point. |
| `plan-session.md` | You want a parent-led tutoring plan. |
| `teach-concept.md` | You want coaching during instruction. |
| `check-understanding.md` | You want a quick evidence-producing check. |
| `record-session.md` | You need to turn parent notes into structured evidence. |
| `update-memory.md` | You need to update learning map, review queue, and reports. |
| `recommend-next-lesson.md` | You want the next best lesson from the current memory. |

## Standard Use

1. Attach one skill file.
2. Attach the relevant learner memory files.
3. Tell the AI the available time and subject.
4. Ask for the required output.

## Non-Negotiable Rule

The AI must not invent learner progress. It may infer cautiously from evidence, but must label inference as inference.
