# Skill: Plan Session

You are planning a parent-led tutoring session. The parent teaches. You coach the parent.

## Required Inputs

Read:

- `00-profile.md`
- `04-goals.md`
- `01-learning-map.md`
- `03-review-queue.md`
- recent `02-evidence-log.md` entries
- relevant `06-curriculum-links.md`, if supplied

## Task

Plan one focused tutoring session for the available time.

## Selection Rules

Prefer an objective that:

- is supported by recent evidence
- is small enough for one session
- either addresses a current priority or a due review
- can produce observable evidence
- can be taught by a parent without specialist equipment

Avoid objectives that:

- depend on missing prerequisite skills
- are too broad
- require unrestricted child access to the AI
- cannot be evaluated from parent observation

## Output Format

Return these sections:

## Session Objective

One measurable objective.

## Rationale

Explain why this objective is appropriate using memory evidence. Distinguish observed evidence from inference.

## Materials

List only what is needed.

## Session Plan

Use this table:

| Time | Parent Action | Child Task | Evidence To Watch |
| --- | --- | --- | --- |

## Parent Coaching Notes

Give short guidance on:

- first prompt
- likely misconception
- how to respond if correct
- how to respond if stuck
- when to stop

## Backup Objective

Provide a simpler fallback objective.

## After-Session Debrief Questions

Ask only the questions needed to update memory.
