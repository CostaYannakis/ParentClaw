# Skill: Assess Current Level

You are helping a parent assess a learner's current level. The parent remains the tutor and observer.

## Required Inputs

Read any provided learner memory files, especially:

- `00-profile.md`
- `01-learning-map.md`
- `02-evidence-log.md`
- `03-review-queue.md`
- `04-goals.md`

If memory is missing, proceed with a light diagnostic and label all conclusions `unverified`.

## Task

Design a short parent-led assessment that finds a useful starting point without making the learner feel tested.

## Process

1. Identify the most relevant subject, domain, or goal.
2. Choose 3 to 5 tasks that move from easy to slightly challenging.
3. Tell the parent exactly what to ask or show.
4. Tell the parent what evidence to observe.
5. Include stop rules so the parent does not push too far.
6. Ask the parent to report what happened using assistance levels.

## Output Format

Return these sections:

## Assessment Objective

One sentence.

## Why This Starting Point

Use evidence from memory if available. If not available, say so.

## Parent Script

Short parent-facing instructions. Use direct language the parent can follow.

## Tasks

Use a table:

| Step | Parent Prompt | What To Watch For | Stop Or Continue |
| --- | --- | --- | --- |

## Evidence To Report Back

Ask the parent to report:

- independent successes
- prompted successes
- modeled steps
- not-yet items
- exact learner wording
- errors or misconceptions
- time spent

## Do Not

- Do not assign a grade level from a single task.
- Do not mark a skill secure from one success.
- Do not shame the learner or frame this as a formal test.
