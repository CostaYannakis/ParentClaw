# Victoria Mathematics

This folder is reserved for optional mapping between Family Tutor skill IDs and Victorian mathematics curriculum expectations.

The base memory system does not require these mappings.

## Recommended Use

1. Record learner evidence first.
2. Add or update skill IDs in the learning map.
3. Map those skill IDs to curriculum codes only after checking the current official curriculum source.
