# Curriculum

Curriculum mapping is optional.

The learner memory is evidence-first. Curriculum files help translate observed skills into school or regional standards, but they should not become the primary learner record.

## Use Curriculum Files To

- align tutoring with school expectations
- identify gaps
- prepare parent-teacher conversations
- explain why a skill matters

## Do Not Use Curriculum Files To

- mark a learner secure without evidence
- force every session to match a code
- hide uncertainty
- replace the learning map

## Starter Files

| File Or Folder | Purpose |
| --- | --- |
| `mapping-template.md` | Template for mapping Family Tutor skill IDs to standards. |
| `victoria-mathematics/` | Placeholder for Victorian mathematics mappings. |
